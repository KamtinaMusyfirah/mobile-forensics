SIM Cards Forensics

Subcriber Identification Module (SIM), juga dikenal sebagai SIM card berisi data khas yang berbeda dari data yang ditangkap oleh akuisisi forensik dari perangkat itu sendiri.

Subcriber Identification Module (SIM)

Peran utama SIM card adalah untuk memberikan dan membuktikan identitas. Selain itu, SIM card memiliki fungsi lain, yaitu menyediakan memori kecil bagi pengguna untuk menyimpan kontak dan menyimpan log panggilan dan SMS. Kartu ini tersedia dalam berbagai ukuran seperti kartu SIM Mini, Mikro, dan Nano.

Saat ini kartu SIM dikirim dalam satu ukuran yang cocok untuk segala hal, sehingga SIM card Nano dapat diambil dari SIM card Mikro, yang juga dapat dilepas dari SIM card Reguler. Ini dikenal sebagai multi SIM card. Dengan kata lain, bisa mengeluarkan SIM card yang diperlukan untuk telepon.

Semua kartu SIM terdiri dari dua bagian penyimpanan data. Satu bagian digunakan untuk sistem informasi dan yang lainnya digunakan untuk informasi pengguna yang dikunci oleh kode PIN pengguna . Dari bagian pengguna, kontak, log panggilan, dan SMS dapat dipulihkan bahkan entitas yang dihapus.

SIM Architecture

SIM card terdiri dari memori volatile dan non-volatile. Pertama, RAM (Random Access Memory) yang merupakan memori volatile yang digunakan untuk mengeksekusi program dalam SIM card. Di sisi lain, jenis memori non-volatile adalah ROM (Read Only Memory) dan EEPROM (Electrically Erasable Programmable Read Only Memory).

Pohon sistem file kartu SIM memiliki tiga kelas pengidentifikasi, yaitu, file utama, file khusus, dan file dasar. Setiap pengidentifikasi sistem file memiliki header dan tubuh. Header berisi informasi tentang file seperti jenis file, file metadata, dan izin. Sistem operasi mengontrol semua operasi akses file SIM card. Ini berfungsi saat operasi normal seperti membaca, menulis, membuat, dll.

File siklis adalah urutan catatan ketika sistem operasi mencoba mengakses catatan berikutnya setelah catatan terakhir dalam urutan kemudian kembali ke yang pertama. Format yang paling banyak digunakan yaitu transparan. Tidak ada format standar karena hanya urutan byte. Apa saja aplikasi dapat menggunakan format sendiri untuk menyimpan datanya. Namun, file apa pun yang transparan format harus ditautkan dengan informasi kontrol yang mengelola pengaksesan file.

Security

Personal Identification Number (PIN) digunakan untuk melindungi data pelanggan dimana kode ini (4â€“8 digit) digunakan untuk verifikasi. Namun, kode PIN digunakan di atas data pelanggan dan data sistem. Personal Unlocking Key (PUK) juga dikenal sebagai PIN Unblocking Key yang digunakan untuk mereset kode PIN yang terlupa. Di sisi lain, PUK digunakan untuk mengatur ulang kode PIN.

Membuka kunci kode PIN melalui perangkat sangat berbahaya karena kegagalan akan menyebabkan pemblokiran perangkat. Itu sebabnya disarankan untuk melepas SIM card pada awal proses forensik. Sayangnya, tanpa membuka kunci PIN kode, penyidik tidak dapat mengekstrak data pengguna dari sistem file SIM.

Evidance Extraction

Untuk SIM card, kontak, log panggilan, dan pesan SMS disimpan di EF berkas sistem. Untuk menemukan bukti-bukti ini, data yang diekstraksi harus diterjemahkan untuk membantu penyidik untuk membaca bukti yang diekstraksi

  • Contacts

    SIM card mungkin berisi beberapa kontak tertentu yang berbeda dari kontak yang ditemukan di perangkat. Ini karena SIM card memungkinkan pengguna untuk menyimpan kontak langsung di atasnya. Selain itu, perangkat telepon memungkinkan pengguna untuk memilih antara perangkat, SIM card, dan cloud untuk menyimpan kontak.

  • Calls

    Di perangkat ponsel, log panggilan keluar, masuk, dan tidak terjawab disimpan dan hanya panggilan keluar yang disimpan dalam SIM card. Konfigurasi perangkat menentukan apakah log panggilan keluar akan disimpan ke penyimpanan kartu SIM atau tidak. Oleh karena itu, log panggilan harus diekstraksi dari SIM card oleh penyelidik forensik yang dapat memastikan bukti yang berguna. Daftar panggilan keluar dikenal sebagai Last Dialed Number (LDN) atau Last Number Dialed (LND).

  • SMS (Short Message Service)

    Salah satu bukti penting yang dapat diekstraksi dari kartu SIM. Pengguna dapat mengirim dan menerima pesan teks yang berisi hingga 160 karakter bahasa Inggris atau 70 abjad bahasa lainnya. Badan SMS dikodekan menggunakan pengkodean 7-bit khusus yang disebut penyandian GSM 03.38 atau Unicode untuk karakter non-Inggris .
