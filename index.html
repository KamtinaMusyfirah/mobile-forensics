Android Forensics

Saat ini, sangat banyak informasi yang dapat kita temui pada smartphone, misalnya riwayat panggilan, kontak, kalender, SMS, MMS, aktivitas internet, foto, video, GPS location, data yang disimpan di kartu SD, dan masih banyak yang lainnya.

Gambar berikut merupakan struktur dari smartphone yang kita gunakan :

  • Processor, ini dianggap sebagai otak dari smartphone.
  • Storage Memory, smartphone modern saat ini menyediakan berbagai penyimpanan memory pilihan, yaitu memory internal dan memory eksternal. Dan memory internal biasanya dipartisi menjadi dua bagian, yaitu : penyimpanan sistem dan penyimpanan telepon.
  • Sensors, saat ini smartphone telah dilengkapi dengan berbagai sensor seperti accelerometer, kompas digital, gravitasi, giroskop, sensor sidik jari, dan termometer.
  • SIM Cards, Sebagian besar ponsel khususnya ponsel GSM memerlukan SIM untuk mengidentifikasi dan mengautentikasi pelanggan ke penyedia layanan seluler atau operator.
  • Network Connectivity, karena Wi-Fi yang tersedia secara meluas ditambah lagi dengan adanya cloud computing di smartphone menyebabkan aplikasi dan penyimpanan data disedikan sebagai layanan bagi pengguna seluler melalui internet.
  • Cameras and Speakers, smartphone masa kini dilengkapi dengan kamera dan speaker.

Investigasi Forensik Perangkat Seluler

Ponsel dilengkapi dengan banyak teknologi nirkabel, termasuk Wi-Fi, bluetooth dan data seluler. Jika ponsel masih hidup dan terhubung dengan jaringan nirkabel, maka sangat penting untuk mengisolasi ponsel dari jaringan disekitarnya untuk menjaga integritas data yang disimpan di ponsel. Dan juga, ponsel yang terhubung ke komputer, misalnya melalui kabel USB harus segera diputuskan.

Ada tiga aspek untuk investigasi yaitu lokasi penyimpanan data, ekstraksi data, dan analisis data. Ada beberapa metode yang dapat digunakan untuk ekstraksi data, diantaranya yaitu metode akuisisi data dan metode analisis data.

Aspek Untuk Investigasi

  • Lokasi Penyimpanan

    Untuk mengekstrak dan menganalisis data yang tersimpan di smartphone, seseorang harus mengetahui dimana data tersebut tersimpan. Gambar dibawah merupakan standar lokasi penyimpanan pada Android.

    Tingkat atas (merah) mewakili partisi (bagian dari semua partisi); yang kedua dan tingkat ketiga (biru dan hijau masing-masing) mewakili konten yang ditemukan di partisi ini. Partisi "userdata", dipasang di "/ data", berisi semua penyimpanan aplikasi pribadi. Ini adalah konten yang dilindungi, hanya dapat diakses oleh setiap aplikasi.

    Partisi sdcard dapat berisi informasi. Misalnya, gambar/video yang diambil oleh aplikasi kamera smartphone, file yang diunduh, dan yang lainnya. Lokasi ini tidak terlindungi, artinya siapa pun memiliki akses ke konten yang disimpan di sdcard. Ini adalah lokasi lain untuk mencari konten yang relevan secara forensik.

  • Metode Akuisis

    Berikut adalah beberapa metode akuisisi data yang berbeda, termasuk kombinasi metode perangkat keras dan perangkat lunak.

    1. Chip Off

      Metode ekstraksi data ini rumit dan membutuhkan banyak teknis pengetahuan, ketangkasan dan kepercayaan diri untuk benar-benar membongkar perangkat seluler. Kelemahan dari metode ini adalah membongkar smartphone dengan hati-hati menjadi tugas yang memakan waktu. Jika ada batas waktu yang ketat dalam penyelidikan, maka metode ini mungkin tidak cocok. Kelebihanya adalah meskipun smartphone dalam keadaan rusak, dimungkinkan untuk mengambil data dari smartphone tersebut selama flash chip nya tidak rusak.

    2. JTAG (Joint Test Action Group)

      JTAG adalah protokol komunikasi yang sering didukung oleh processor untuk menyediakan akses ke fungsi debug / emulasi. Kelebihan dari metode ini adalah lebih sedikit modifikasi fisik perangkat yang diperlukan, namun CPU tidak boleh rusak agar metode ini dapat dilakukan. Kelemahannya yaitu tidak semua perangkat mendukung JTAG, Koneksi test action port sulit ditemukan, ekstraksinya relatif lebih lambat, terutama ketika mempertimbangkan ukuran penyimpanan internal smartphone modern.

    3. Forensic Software Suites

      Ada banyak forensic Software suites yang dirancang untuk perangkat smartphone. Kelebihan dari metode ini yaitu, tidak hanya menyediakan metode untuk mengekstrak data, tetapi juga menampilkan dan menyimpannya dengan cara yang nyaman. Kelemahannya yaitu software nya sangat mahal, dukungan untuk perangkat baru mungkin agak lambat, karena kebutuhan untuk memperbarui perangkat lunak sesuai.

    4. ADB (Android Debug Bridge)

      Ini adalah alat baris perintah yang memungkinkan komunikasi antara komputer dan perangkat seluler Android (atau emulator). Program client server mencakup: client, daemon, server. Kelebihan dari metode ini adalah mencakup banyak perintah, diantaranya yaitu pull, push, shell dan lain lain. Kelemahannya yaitu untuk berinterkasi dengan perangkat menggunakan metode ini, USB debugging harus diaktifkan dan layar harus dibuka.

    5. Backup Applications

      Mekanisme ini tidak akan menyalin semua data aplikasi, karena hanya memiliki akses ke aplikasi data yang diprogram untuk dibagikan. Keuntungan dari metode ini adalah mudah. Seseorang hanya perlu menginstal aplikasi (dan mungkin kartu microSD), dan biarkan aplikasi melakukan tugasnya. Ada juga kelemahan dari metode ini, yaitu harus memiliki akses ke perangkat (mis . Ini berarti bahwa metode ini tidak akan layak jika Anda tidak memiliki akses melewati layar kunci.

    6. Firmware Update Protocols

      Perangkat dapat di-boot ke pembaruan firmware atau mode unduh, dan ketika dalam mode ini, proses update firmware dapat berjalan, yang digunakan untuk flash baru pada firmware sistem. Ada banyak Kelebihan dari metode ini, yaitu menyediakan cara yang canggih untuk memulihkan informasi dari banyak smartphone, tanpa mempengaruhi integritas data perangkat. gambar fisik dipulihkan, yang menyediakan lebih banyak data daripada gambar logis. Kelemahan dari metode ini, yaitu membutuhkan waktu dan usaha untuk bootloader reverse engineer dan perangkat lunak pembaruan firmware untuk perangkat yang berbeda.

  • Analisis Data

    Setelah mengekstrak data dari telepon, sampailah pada langkah analisis data. Pendekatan analisis data bervariasi dengan aplikasi Android sepert pesan, panggilan telepon, browser web, dan sebagainya. Dengan kata lain, setiap aplikasi membutuhkan metode analisis forensik yang unik. Data dianalisis dan aktivitas atau kebiasaan pengguna diselidiki dari aspek berikut:

    1. Analysis of contact information: Informasi kontak memungkinkan penyelidik untuk menentukan dengan siapa pengguna melakukan kontak.
    2. Analysis of exchanged messages: Kronologi pesan yang dipertukarkan dapat direkonstruksi dengan menentukan stempel waktu dari pesan yang dipertukarkan, data yang dibawa, pengguna yang terlibat dalam percakapan, serta kapan percakapan itu benar-benar telah diterima oleh penerimanya.
    3. Analysis of deletions: penghapusan ini memberikan beberapa panduan untuk penyelidikan.
